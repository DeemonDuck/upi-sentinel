# Assets – Project Visualizations

This folder contains visual outputs generated from the final evaluation of the UPI Sentinel fraud detection models.

These visualizations are used in the main README and demonstrate:

- Confusion Matrix (Best Threshold)
- ROC Curve
- Precision-Recall Curve
- CRNN Architecture
- Sliding Window (Illustration)

All images correspond to the final Hybrid CNN + BiLSTM + Attention model evaluated on the imbalanced test dataset.

These visuals support performance interpretation and threshold optimization analysis.
